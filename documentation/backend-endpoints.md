# API Endpoints

## Users

### User authentication

- POST /api/session

### User account creation

- POST /api/users

## Beats

### Create a beat (music)

- POST /api/beats

### Get all beats

- GET /api/beats

### Get a beat

- GET /api/beats/id

## Room

### Create a room

- POST /api/room

### Delete a room

- DELETE /api/room

## Mixes (Stretch Goal)

### Create a mix

- POST /api/mix

### Get all mixes

- GET /api/mix

### Search for a mix

- GET /api/mix/search
