# API Endpoints

## Users

### User authentication
- POST /api/users/token
### User account creation
- POST /api/users/create

## Beats

### Create a beat (music)
- POST /api/beats
### Get all beats
- GET /api/beats

## Session

### Create a session
- POST /api/session
### Delete a session
- DELETE /api/session

## Mixes (Stretch Goal)

### Create a mix
- POST /api/mix
### Get all mixes
- GET /api/mix
### Search for a mix
- GET /api/mix/search
